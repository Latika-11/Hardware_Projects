# Alcohol Detection and Engine Locking System

## Overview
This project is designed to improve road safety by detecting alcohol consumption in drivers and preventing vehicle ignition when alcohol is detected.

## Objective
The objective of this project is to reduce drunk driving accidents using alcohol sensing technology.

## Components Used
- MQ-3 Alcohol Sensor
- Arduino Uno
- Relay Module
- Buzzer
- LED
- DC Motor
- Connecting Wires
- Power Supply

## Working Principle
The MQ-3 sensor detects alcohol concentration from the driver's breath. If alcohol exceeds the threshold value, the relay disables engine ignition and activates the warning system.

- Vehicle Safety Systems
- Drunk Driving Prevention
- Smart Transportation Systems
